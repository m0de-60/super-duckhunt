<H2>VERSION & DEVELOPMENT NOTES:</H2>

4-27-2026 Fixed a bug that was causing crashed due to duplicate key entries in duckhunt.cnf. This fix resulted in reworking a lot of the config and splitting the configuration values into multiple files. Every instance running will have duckhunt_servername_channel_name.cnf where config sections [servername_channelname] and [servername_channelname_ducks] will now be located. Each server and channel now has its own config file that is self generated by the script, and requires no additional new files to be installed. duckhunt.cnf will only be used for thread identification for the channels and servers where duckhunt will run on the bot. Configuration will be done through the bot, similar to how it is done in version 1.1.4. Also made some more changes to sys_zcore, so zCore update will be required as well.
